# shecannotsee.github.io

个人博客仓库，对应：

- 仓库：<https://github.com/shecannotsee/shecannotsee.github.io>
- 页面：<https://shecannotsee.github.io>

## 当前结构

- 首页数据：`_data/site_profile.yml`
- 首页布局：`_layouts/home.html`
- 公共导航与页脚：`_includes/site_nav.html`、`_includes/site_footer.html`
- 全站样式：`assets/css/main.scss`
- 主题/语言切换：`assets/js/site_preferences.js`
- 文章：`_posts/`
- 维护文档：`how_to_modify_this_project.md`
